Branches:
	- Master
	- Development
	- Finalization
	- Feature
	
	
Master:
	Final releases of different versions, all versions here will be fully working and all commits will be merges with Finalization.
	
	
Finilization:
	When getting ready to release a new version, Development is merged to Finalization and from there no new featueres are implemented on this branch and all features are made sure they work as the requirements to the release describe.
	
	
Development:
	When working on a already existing feature it is done in this branch, when it's almost ready for release it is merged into Finalization.
	
	
Feature:
	When a new feature needs to be made the package, classes or meathods are created in this branch, and when they are working they are pushed to development.
